	Berikut adalah roadmap pengerjaan kita. Kita akan eksekusi satu per satu setelah bos konfirmasi setiap tahapnya:
	
	Tahap 1: Pondasi & Database (The Foundation)
		Setup .env untuk koneksi PostgreSQL.
		Konfigurasi AppModule untuk menghubungkan TypeORM.
		Setup main.ts (Global Prefix, Validation Pipe, dan Static Folder).
		Membuat Role Enum (SUPER_ADMIN, STAFF).
	
	Tahap 2: Manajemen User (User Identity)
		Membuat User Entity (ID, Username, Password, Role).
		Membuat UsersService untuk logika pencarian dan pembuatan user (Admin).
		Membuat Initial Seeder (Script otomatis buat akun Super Admin pertama kali).
	
	Tahap 3: Keamanan & Autentikasi (The Gatekeeper)
		Implementasi Bcrypt untuk hash password.
		Setup JwtStrategy untuk validasi token.
		Membuat AuthService untuk logika login (Username + Password -> JWT).
		Membuat AuthGuard dan RolesGuard (Proteksi khusus Super Admin).
	
	Tahap 4: Modul Menu (Core Business)
		Membuat Menu Entity (Nama, Harga, Stok, Kategori FOOD/DRINK, Image Path).
		Membuat CRUD Menu di MenuService.
		Integrasi Multer di MenuController untuk Image Upload.
		Validasi input menggunakan DTO (Data Transfer Object).
	
	Tahap 5: Testing & Finishing
		Uji coba endpoint via Postman/Insomnia.
		Memastikan relasi data dan proteksi role berjalan (User biasa tidak bisa hapus menu).
